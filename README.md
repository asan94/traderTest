# Тестовое задание

Реализовать основную таблицу с информацией по котировкам (только один экран).  
Данные обновляются в реальном времени, подписка на котировки по сокетам.

Аналогично главному экрану уже существующего приложения  
Tradenet by Freedom Finance

- App Store: https://apps.apple.com/app/id1178912301  
- Google Play: https://play.google.com/store/apps/details?id=com.tradenet.terminal  
- Мобильная версия сайта: https://tradernet.com/terminal/mobile/quotes  

Отнестись к выполнению тестового как к задаче от заказчика.  
Необходимы как корректная работа бизнес‑логики, так и соответствие интерфейса заказанному.

## Тезисы для выполнения задания

### Взаимодействие с бек‑ендом

1. Авторизация не требуется.  
2. Сокетное подключение для данных по котировкам.  

   - Сервер: `wss.tradernet.com`  
   - Событие на подписку `quotes`, например:  
     `["quotes", ["AFLT", "AAPL.US"]]`  
   - Данные приходят в ответе `q`.  
   - Описание структуры ответа указано здесь:  
     https://tradernet.com/tradernet-api/quotes-get-changes  

3. Использовать фиксированный список бумаг с идентификаторами:  

   `SP500.IDX,AAPL.US,RSTI,GAZP,MRKZ,RUAL,HYDR,MRKS,SBER,FEES,TGKA,VTBR,ANH.US,VICL.US,BURG.US,NBL.US,YETI.US,WSFS.US,NIO.US,DXC.US,MIC.US,HSBC.US,EXPN.EU,GSK.EU,SHP.EU,MAN.EU,DBI.EU,MUV2.EU,TATE.EU,KGF.EU,MGGT.EU,SGGD.EU`

4. Данные, используемые в верстке:

   1. Тикер  
   2. Изменение в процентах относительно цены закрытия предыдущей торговой сессии  
   3. Биржа последней сделки  
   4. Название бумаги  
   5. Цена последней сделки  
   6. Изменение цены последней сделки в пунктах относительно цены закрытия предыдущей торговой сессии  

5. Значение с положительным значением имеет зеленый шрифт,  
   с отрицательным — красный.  

6. Подсветка идет зеленым при изменении значения в положительную сторону,  
   красным — при изменении значения в отрицательную сторону.  

7. Разрешено использование сторонних сетевых библиотек.  

8. Требования iOS:

   - Верстка UIKit + SnapKit  
   - Архитектура MVC, MVVM или VIPER  
   - iOS начиная с 15.0  

9. Требования Android:

   - Верстка Compose  
   - Архитектура MVVM  

## Дополнительно (по желанию)

1. Получить список котировок в API:  
   https://tradernet.com/tradernet-api/quotes-get-top-securities  
   с параметрами:  
   - `type`: `stocks`  
   - `exchange`: `russia`  
   - `gainers`: `0`  
   - `limit`: `30`  

2. Отобразить логотип компании:  
   `https://tradernet.com/logos/get-logo-by-ticker?ticker=` + *lowercased ticker*  

3. Округлить выводимое значение цены и изменения цены до `min_step`.
